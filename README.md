# 🔐 Лабораторная работа: Шифры Виженера и Плейфера

## 📋 Описание проекта
Веб-приложение для шифрования и дешифрования текста с использованием двух классических криптографических алгоритмов:
- **Шифр Виженера** (для русского языка, 33 буквы)
- **Шифр Плейфера** (для английского языка, 4 матрицы)

## ✨ Возможности

### 🎯 Основной функционал
- ✅ Шифрование и дешифрование текста
- ✅ Поддержка русского языка для Виженера
- ✅ Поддержка английского языка для Плейфера (I=J)
- ✅ 4 матрицы для шифра Плейфера
- ✅ Визуализация таблицы Виженера
- ✅ Отображение всех 4 матриц Плейфера

### 📁 Работа с файлами
- ✅ Загрузка текста из .txt файла
- ✅ Сохранение результата в .txt файл
- ✅ Копирование результата в буфер обмена

### 📊 Интерфейс
- ✅ Статистика текста (длина, количество букв)
- ✅ Индикация текущего языка
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Подсветка ячеек в таблице Виженера
- ✅ Пример шифрования с детализацией
